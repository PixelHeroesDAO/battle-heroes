# Battle Heroes

Battle Heroes is a game based on Pixel Heroes NFT collections.

## Project setup

```sh
yarn install
```

## Backend

### Start development server

```sh
yarn dev
```

### Start production server

```sh
yarn start
```

## Frontend

### Compiles and hot-reloads for development

```sh
yarn serve
```

### Compiles and minifies for production

```sh
yarn build
```
